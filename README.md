# Gadget Hub

Loja de dropshipping de eletrónicos, casa inteligente e acessórios tech.

**Site:** gadget-hub.com  
**Plataforma:** Shopify + CJ Dropshipping  
**Stack:** HTML/CSS/JS (protótipo) → Shopify (produção)

## Estrutura do Projeto

- `index.html` — Página principal da loja
- `produto.html` — Página de produto individual
- `checkout.html` — Página de checkout
- `sistema/` — Sistema de skills automatizadas
  - `orquestrador.js` — Orquestrador central
    - `app.js` — Ponto de entrada
      - `skills/skill1-produtos.js` — Pesquisa e score de produtos
        - `skills/skill2-site.js` — Gestão de páginas
          - `skills/skill3-marketing.js` — Copy e anúncios
            - `skills/skill4-automacao.js` — Chatbot e automação
            - `GUIA-APIs-E-PAGAMENTOS.md` — Guia de APIs e pagamentos
            - `GUIA-GADGET-HUB-COMPLETO.md` — Guia completo do projecto

            ## Nicho

            Eletrónicos, Casa Inteligente & Acessórios Tech para o mercado português e ibérico.
